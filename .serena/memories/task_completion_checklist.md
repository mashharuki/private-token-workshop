# タスク完了時のチェックリスト

## 開発完了時の必須チェック
1. **コンパイルチェック**:
   ```bash
   cd token_template
   leo build
   ```

2. **テスト実行**:
   ```bash
   leo test
   ```

3. **構文チェック**:
   ```bash
   leo check
   ```

4. **プログラム設定の確認**:
   - `src/main.leo`のプログラム名と`program.json`の"program"フィールドが一致しているか
   - プログラム名が10文字以上であるか

## Git操作前のチェック
1. **変更の確認**:
   ```bash
   git status
   git diff
   ```

2. **コミット前のチェック**:
   - 適切なコミットメッセージ（コンベンショナルコミット形式）
   - 関連のない変更が混在していないか

## デプロイ前のチェック
1. **テストネット準備**:
   - テストネットクレジットの確保
   - プログラム名の重複チェック

2. **最終確認**:
   - すべての必要な機能が実装済みか
   - コンプライアンスチェックが正しく動作するか

## トラブルシューティング
- ビルドエラー: プログラム名の不一致をチェック
- テストエラー: 関数シグネチャの確認
- デプロイエラー: ネットワーク接続とクレジット残高の確認
